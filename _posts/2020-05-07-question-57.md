## 문제 57 : 1의 개수

```js
// 0부터 1000까지 1의 개수를 세는 프로그램을 만들어라.
// 11은 1이 두개, 111은 1이 세개이다.
```

## 풀이

```js
let s = ``;
let count = 0;
for (let i = 0; i <= 1000; i++) {
  s += i;
}
for (j = 0; j < s.length; j++) {
  if (s[j] === "1") {
    count++;
  }
}

console.log(count);
// 301
```
