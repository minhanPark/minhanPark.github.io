# 데이터 불러오기 개요

Next.js에서 데이터 불러오기는 애플리케이션의 사용방법에 따라 여러 방식으로 컨텐츠를 렌더링하도록 도와줍니다.  
여기에는 Server-side 렌더링 또는 Static Generation 등의 미리 불러오기와 런타임에서 컨텐츠 수정과 생성이 가능한 Incremental Static Regeneration을 포함하고 있습니다.
